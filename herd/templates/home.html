A well-annotated database of putative and validated human enhancers.

The database contains putative and validated human enhancers in the hg38 assembly collected and curated from ENCODE's DNase-seq expeirment dataset. The Enhancers' data has been compared with and annotated using Genomic data provided by Reference Genome Annotation and validated Enhancer data provided by VISTA and eRNA datasets. Enhancers in the database have been taxonomized based upon whether they overlap Genomic Features such as genes and promoters, and whether they overlap with enhancers found in the VISTA and eRNA datasets. Currently the database has a total of 104555 annotated enhancers curated from 1045 DNase-seq experiments from ENCODE.

In version 1 of HERD you are able to query the database based on chromosome, a specifc region within the chromosome, tissue type and other specifications. To learn more about how you can query HERD please see Help.
